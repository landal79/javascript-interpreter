JS SIMULATOR 1.0

Questo documento costituisce la documentazione del sistema software JS Simulator 1.0. Questo sistema rappresenta un ambiente di simulazione per il core linguaggio Javascript, in pratica inserendo nel dispositivo di ingresso dei programmi scritti in Javascript si riceverà in uscita l'elenco delle variabili contenute nell'ambiente globale, contenenti i valori che risultano dall'elaborazione. Il software è organizzato in diversi package contenenti le diverse parti che realizzano le varie funzionalità di cui si compone il sistema.
