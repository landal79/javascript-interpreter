Gerarchia APT

  • La gerarchia APT contiene le possibili categorie sintattiche di cui è composta una frase di javascript; le categorie sintattiche possono essere desunte dalla grammatica generale, togliendo tutte le informazioni accessorie, si ottiene così la grammatica dell'apt (vedi grammatiche), nella quale sono presenti le varie categorie sintattiche del linguaggio. Ogni categoria sintattica corrisponde ad una classe concreta della gerarchia, ogni classe è caratterizzata dal fatto di avere diversi campi, uno per ogni sottotipo (si intende categoria sintattica) che compone la regola sintattica.
  • Con questa gerarchia si costruisce l'albero di derivazione delle frasi del linguaggio, esso costiuisce la rappresentazione interna della frase usata usata dai visitor per compiere le possibili valutazioni di una frase; sarà il parser a creare l'albero di derivazione come risultato dell'analisi sintattica.
  • Categoria sintattica fondamentale è l'istruzione dalla quale derivano tutte le altre in quanto sono particolari istruzioni.
