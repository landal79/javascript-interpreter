Introduzione sui linguaggi

  • Un linguaggio di programmazione, per essere ben definito deve rispettare delle regole per la composizione delle frasi, in questo modo è possibile stabilire se una determinata frase appartiene o meno ad un linguaggio.

    •  
  • Una generica frase di un linguaggio oltre ad essere ben costruita secondo le regole grammaticali, deve essere avere un significato valido; questi due aspetti sono rispettivamente:
    • sintassi: è l'insieme delle regole formali per la scrittura di programmi in un linguaggio;
    • semantica: è l'insieme dei significati da attribuire alle frasi (sintatticamente corrette) costruite nel linguaggio;

      •  
  • Una frase viene analizzata su tre diversi livelli, lessicale, sintattico e semantico:

    • Analisi Lessicale: consiste nell'individuazione delle singole parole (token) che compongono una frase; I token che possono comparire nelle frasi di un linguaggio sono stabiliti dalle regole grammaticali che descrivono il linguaggio;

      •  
    • Analisi Sintattica: consiste nella verifica che una frase possa essere costruita in base alle regole grammaticali che descrivono il linguaggio.

      •  
    • Analisi Semantica: consiste nel "calcolo" del significato del linguaggio;

    •  
  • Facciamo ora una precisazione sul modo in cui possono essere analizzata i linguaggi, distinguiamo fra Interpreti e Compilatori:

    • Interprete: per un linguaggio L, è un programma che accetta in ingresso le frasi di L e le esegue una per volta; l'output dell'interprete è quindi una valutazione della frase;

      •  
    • Compilatore: per un linguaggio L, è un programma che accetta in ingresso un programma scritto in L e lo riscrive in un altro linguaggio; l'output di un compilatore è una riscrittura dell'intero programma in un altro, purchè il primo sia sintatticamente e semanticamente corretto;

Definizione di Grammatica

  • Alfabeto: un alfabeto A è un insieme di simboli finito e non vuoto di simboli atomici;
  • Stringa: una stringa è una sequenza si simboli, ossia un elemento del prodotto cartesiano A x A x ... x A;
  • Linguaggio L su un alfabeto A
    • un linguaggio L è un insieme di stringhe su A;
    • Frase (sentence) di un linguaggio: una stringa appartenente a quel linguaggio;
    • Cardinalità di un linguaggio: il numero delle frasi del linguaggio;
  • Chiusura A* di un alfabeto: è l'insieme infinito di tutte le stringhe composte con simboli di A;
  • Chiusura positiva A+ di un alfabeto A: è l'insieme infinito di tutte le stringhe non nulle composte con simboli di A;

    •  
  • Una Grammatica è una notazione formale con cui esprimere in modo rigoroso la sintassi di un linguaggio.

    •  
  • La sintassi di un linguaggio può essere espressa attraverso la notazione BNF, così definita:
      Una grammatica è una quadrupla <VT, VN, P, s> dove:
    • VT è un insieme finito di simboli terminali
    • VN è un insieme finito di simboli non terminali
    • P è un insieme finito di produzioni ossia di regole di riscrittura α->β α e β sono stringhe
    • S è un particolare simbolo non-terminale detto simbolo iniziale o scopo della grammatica.
