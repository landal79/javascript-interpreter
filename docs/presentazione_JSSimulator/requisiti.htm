Elenco dei requisiti

Dalle specifiche concordate con l'utente il sistema dovrà supportare le seguenti funzionalità:

  • La parte del linguaggio di interesse è il core di javascript, nella parte che è stata descritta nel relativo glossario;

    •  
  • Possibilità di eseguire l'analisi sintattica della frase, fornendo in uscita l'indicazione di successo nel caso la frase sia sintatticamente corretta e la riproduzione della frase sul dispositivo di uscita per dimostrare che è stata correttamente riconosciuta, altrimenti fornendo un opportuna indicazione di errore;

    •  
  • Fornire una interpretazione della frase o programma in ingresso (Interprete di javascript), eseguita secondo le regole semantiche del linguaggio javascript, come indicato nella versione 1.5; nel caso di successo verrà fornita in uscita la lista delle variabili con i valori che contengono, altrimenti verrà visualizzata la relativa segnalazione di errore;
    •  
     
  • L'ambiente di simulazione opererà nel seguente modo: sarà possibile richiedere una delle operazioni sopra indicate sulla frase in ingresso sul dispositivo di input, quando verrà richiesta una nuova operazione l'ambiente verrà reinizializzato, così l'esecuzione di ogni operazione è indipendente;
