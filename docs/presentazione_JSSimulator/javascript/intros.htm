Javascript

  • Javascript è un linguaggio di programmazione interpretato e leggero, che dispone di funzionalità object-based. Tale linguaggio è nato espressamente per il web, infatti il suo nucleo è stato incorporato da tutti i principali browser in commercio. La versione di javascript lato client permette di inserire contenuti eseguibili all'interno di pagine web, consentendo di ottenere pagine HTML dinamiche, che interagiscono con l'utente. La versione che verrà analizzata è Javascript 1.5.
  • Sintatticamente, il linguaggio di base assomiglia alla famiglia C, C++, Java. Dispone di costrutti di programmazione tipici di tale famiglia, come le istruzioni di selezione e i cicli. Le caratteristiche principali di tale linguaggio differiscono radicalmente da quelle di java, infatti: Javascript è un linguaggio di programmazione debolmente tipizzato.
