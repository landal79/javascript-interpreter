Creazione di Array

  • Un elemento di un array javascript può essere di qualsiasi tipo e differenti elementi dell'array possono essere di tipi diversi. Gli elementi degli array possono persino contenere altri array: ciò consente di creare strutture di dati formati di array di array. Gli elementi negli array sono assegnati a partire dall'elemento di posizione 0.

    •  
  • Ci sono 3 modi in javascript per creare array, usando con il costruttore Array e l'operatore new (per una panoramica complessiva viene spiegata anche questa possibilità, ma non verrà in effetti inclusa nel progetto):
    • creazione di un array vuoto, esempio:
      var a = new Array( );

      •  
    • specificando i primi n elementi dell'array, esempio:
      var a = new Array (5,4,3,2,1,"testing,testing");

      •  
    • specificando un singolo argomento numerico che indica il numero di elementi, esempio:
      var a = new Array(10);

    •  
  • Possiamo creare un array con un letterale array, inserendo un elenco di valori separati da virgole tra parentesi quadre:
      var primi = [2,4,6,8,11];
      var a = ['a',true,5.77];

Lettura e scrittura degli elementi di un Array

  • Per accedere agli elementi di un array si utilizza l'operatore [ ]. A sinistra delle parentesi deve comparire un riferimento all'array. All'interno delle parentesi deve trovarsi un'espressione arbitraria di valore non negativo. Tale sintassi può essere utilizzata sia per leggere, sia per scrivere il valore di un elemento in un array.

    •  
  • Il primo elemento degli array in javascript hanno indice 0.

Aggiunta di nuovi elementi a un array

  • In javascript, un array può avere un numero qualsiasi di elementi e tale numero può essere modificato in qualsiasi momento, cioè la lunghezza di un array può essere modificata dinamicamente.

    •  
  • per aggiungere un elemento ad un array basta semplicemente assegnare un valore.
