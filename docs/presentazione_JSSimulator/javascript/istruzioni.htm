Definizione di istruzione

  • Le istruzioni in javascript rappresentano qualcosa di simile a una frase completa o a un comando, esse possono contenere espressioni oppure le espressioni possono essere utilizzate all'interno delle strutture di controllo.

    •  
  • Le espressioni devono essere separate una dall'altra da un punto e virgola (;) oppure da un fine riga.

    •  
  • Il tipo più semplice di istruzione di javascript è rappresentato dalle espressioni che producono degli effetti.

    •  
  • Le istruzioni di assegnamento rappresentano la principale categoria all'interno delle espressioni.

    •  
  • Si può definire anche un blocco di istruzioni usando le parentesi graffe { }, all'interno si mettono le istruzioni.

Strutture di controllo

  • if

    if (espressione)
       istruzione

    In questa forma viene valutata l'espressione: se il valore risultante è vero (true) viene eseguita l'istruzione specificata, mentre se l'espressione è falsa (false), l'struzione non viene eseguita

    if (espressione)
       istruzione1
    else
       istruzione2

    Se l'espressione è vera viene eseguita l'istruzione1 altrimenti l'istruzione2

  • switch
    sintassi:

    switch (espressione) {
       istruzioni
    }

  • while
    sintassi:
    while (espressione)
       istruzione

    viene ripetuta l'istruzione finchè l'espressione è vera

  • do/while
    sintassi:
    do
       istruzione
    while (espressione);

    viene ripetuta l'istruzione finchè l'espressione è vera, ma a differenza del while l'istruzione viene eseguita almeno una volta.


  • for
    sintassi:
    for (inizializzazione; test; incremento)
       istruzione
    è analogo a:
    inizializza;
    while (espressione)
       istruzione
       incremento;


  • for/in
    sintassi:
    for (variabile in oggetto)
       istruzione

    variabile rappresenta il nome di una variabile, oggetto è il nome di un oggetto, oppure una espressione che viene valutata come un oggetto. Con questa struttura di controllo vengono lette una per una tutte le proprietà di un oggetto senza conoscerne il nome.

  • break
    sintassi:
    break;

    l'istruzione break fa in modo di uscire immediatamente da un ciclo più interno o da un'istruzione switch.


  • return
    sintassi:
    return espressione;

    tale istruzione serve a ritornare un valore da una istruzione.

    La sintassi è la seguente:

    var nome_1 [ = valore_1] [,..., nome_n [=valore_n]
