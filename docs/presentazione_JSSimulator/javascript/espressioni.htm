Espressioni e operatori

  • Una espressione è una "frase" di codice Javascript da cui l'interprete è in grado di estrarre un valore. Le espressioni più semplici sono letterali o nomi di variabili:

    •  
    1.7 // letterale numerico
    "Javascript è bello!" // letterale stringa
    true // letterale booleano
    { x:2, y:2} // letterale oggetto
    [4,5,6,7,8,9] // letterale array
    function(x){return x*x;} // letterale funzione
    i // variabile i

  • Operatore di creazione degli oggetti

      • L'operatore new crea un nuovo oggetto e richiama una funzione costruttore per inizializzarlo. E' un operatore unario e precede il suo operando. Ha la seguente sintassi:

        new construttore (argomenti)

        dove costruttore dev'essere un'espressione di chiamata di funzione costruttore, ed è seguito da un elenco di zero o più argomenti racchiusi fra parentesi tonde. L'operatore new crea per prima cosa un nuovo oggetto senza alcuna proprietà. Quindi richiama la funzione costruttore specificata passandole gli argomenti indicati e l'oggetto appena creato sotto forma di valore della parola chiave this. La funzione costruttore è quindi in grado di usare l'oggetto di riferito da this inizializzandolo secondo necessità.
