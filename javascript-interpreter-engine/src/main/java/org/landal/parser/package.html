Questo package contiene i tipi e le rispettive classi implementanti necessarie per eseguire l'analisi sintattica di una frase di javascript. In Particolare viene definito il parser definito dal tipo IParser, implementato dal tipo Parser, poi viene definita l'interfaccia LLAnalisys che definisce le funzionalità necessarie per eseguire l'analisi LL(k) di una frase.
